# Prompts Padrão — Nexus

---

## PROMPT PRINCIPAL (usar sempre ao iniciar sessão com IA)

```
Leia SYSTEM.md, MASTER.md e RESUMO.md antes de responder.
Use o mínimo de tokens possível.
Evite repetir contexto já dado.
Responda direto ao ponto.
Se aprender algo relevante novo, atualize a memória.
```

---

## Criação de copy

```
Contexto: infoproduto low-ticket para [AVATAR] no nicho [NICHO].
Produto: [NOME] — [BENEFÍCIO PRINCIPAL]
Preço: R$[VALOR]

Crie:
1. Headline (número + benefício + sem objeção principal)
2. Sub-headline (prova social + acesso imediato)
3. 3 bullet points de benefícios (emocionais, não técnicos)
4. CTA com urgência (máx 8 palavras)

Tom: direto, empático, sem jargões.
Idioma: PT-BR.
Máx 150 palavras no total.
```

---

## Geração de ideias de produto

```
Nicho: [NICHO]
Público: [AVATAR]
Objetivo: gerar 5 ideias de infoproduto low-ticket (R$17–R$47)

Para cada ideia retornar:
- Título (headline pronta)
- Formato (ebook/checklist/planner/workbook)
- Dor que resolve
- Argumento de venda em 1 frase

Priorizar: dores urgentes, resultado rápido, sem conhecimento prévio necessário.
```

## Melhoria de oferta

```
Oferta atual:
Produto principal: [PRODUTO] — R$[PREÇO]
[Descreva o que está incluso]

Analise como especialista em conversão low-ticket Brasil.
Retorne:
1. O que está fraco (máx 3 pontos)
2. Sugestão de order bump (produto complementar + preço)
3. Sugestão de upsell 1 (versão avançada + preço)
4. Nova headline para o produto principal
5. Uma mudança que sozinha aumentaria conversão

Seja específico. Sem rodeios.
```
