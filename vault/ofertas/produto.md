# Produtos e Ofertas

## Nexus Forge — Gerador de Infoprodutos (ferramenta interna)
- **URL:** https://claude-code-febra.onrender.com/criar
- **Função:** gera ebooks, workbooks, carrosséis, criativos com IA
- **Tipos:** Ebook · Workbook · Checklist · Planner · Script VSL · Cheat Sheet · Certificado
- **Output:** PDF + DOCX (PDFKit local) ou link Gamma AI
- **Status atual:** Gamma retry fix deployado (2026-05-04)

## Estrutura de oferta padrão (low-ticket)
```
Produto principal: R$17–R$47
  └── Order Bump: R$7–R$17 (complementar)
       └── Upsell 1: R$47–R$97 (versão avançada)
            └── Upsell 2: R$97–R$197 (mentoria/comunidade)
```

## Copy de produto que funciona
- Headline: número + benefício + sem objeção
  Ex: "7 Segredos do Emagrecimento sem Academia ou Dieta"
- Sub: prova social + acesso imediato
  Ex: "+3.200 mulheres transformadas · baixe agora"
- CTA: ação clara + urgência
  Ex: "Quero meu ebook agora — R$17"
