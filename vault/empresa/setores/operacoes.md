# Setor 7 — Operações (COO)

**Responsável:** MAX (executa tudo automaticamente)
**Presidente:** Rodrigo Cruz (aprova apenas)

---

## Missão

Operar a empresa de ponta a ponta sem que o Presidente precise executar nada.
MAX é o COO. O Presidente é o chairman: vê o resumo, aprova ou veta.

---

## O que este setor faz

| Função | Frequência | Ferramenta |
|--------|-----------|------------|
| Rodar o ciclo semanal completo (7 etapas) | Semanal | `empresa/ciclos/ciclo_NNN.md` |
| Gerar ideias de produto | Semanal | Análise de nicho + backlog |
| Criar oferta completa | Por produto aprovado | Framework monetização |
| Gerar copies prontas | Por produto | copywriter.js + regras/copy.md |
| Gerar roteiros de vídeo | Por produto | roteirista.js |
| Atualizar memória do sistema | Após cada ciclo | estado_atual.md + decisoes.md |
| Identificar melhorias UX/técnicas | Semanal | Checklist UX |

---

## Como o Presidente usa este setor

```
1. Abrir o arquivo do ciclo atual: empresa/ciclos/ciclo_NNN.md
2. Ler cada seção — MAX já fez todo o trabalho
3. Aprovar ou vetar cada entrega com um comentário simples
4. MAX registra a decisão em conselho/decisoes.md
5. Próximo ciclo começa na semana seguinte
```

**Regra de ouro:** O Presidente nunca produz conteúdo. Apenas aprova.

---

## Arquivos de ciclo

Cada ciclo gera um arquivo em `empresa/ciclos/`:

```
ciclo_001.md — Semana 2026-05-04 (primeiro ciclo)
ciclo_002.md — Semana 2026-05-11
ciclo_003.md — Semana 2026-05-18
...
```

Cada arquivo contém:
- 3 ideias de produto (Presidente escolhe 1)
- Oferta completa do produto escolhido
- Copies prontas para usar
- Roteiro de Reels pronto para gravar
- 1 melhoria UX sugerida
- 1 automação sugerida
- Memória atualizada
