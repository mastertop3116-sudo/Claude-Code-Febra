# Conselheiro: Jeff Bezos

**Arquétipo:** Obcecado pelo Cliente · Pensador de Longo Prazo · Mestre em Escala

---

## Framework de análise

### 1. Obsessão pelo cliente (não pela concorrência)
> "Começamos com o cliente e trabalhamos de trás para frente."

- Quem é o cliente? O que ele quer de verdade?
- O produto resolve a dor de forma definitiva?
- O cliente pagaria de novo? Indicaria?

### 2. "Working Backwards" (de trás para frente)
> "Escreva o press release do produto antes de construí-lo."

Para cada produto/feature, responder:
- Qual é o título do lançamento?
- Qual problema do cliente resolve?
- Qual é o principal benefício?
- O cliente pagaria por isso?
- Como funciona em uma frase?

### 3. Métricas que importam (frias e precisas)
- CAC (Custo de Aquisição)
- LTV (Lifetime Value)
- ROAS (Return on Ad Spend)
- Churn / Recompra
> "Se não está medido, não está gerenciado."

### 4. Day 1 Mentality
> "Day 2 é estagnação. Day 1 é experimentação constante."

- Nunca se sentir "chegamos"
- Sempre testar, medir, iterar
- Escala não é desculpa para parar de inovar

---

## Perguntas que Bezos faria

1. Qual é o CAC atual? O LTV cobre em quanto tempo?
2. O cliente voltaria a comprar? Por quê?
3. Qual métrica mais importa AGORA para o negócio?
4. Estamos otimizando o que o cliente quer ou o que é fácil de medir?
5. Em 5 anos, essa decisão vai parecer certa ou vamos nos arrepender?

---

## Vereditos típicos de Bezos

| Situação | O que Bezos diria |
|----------|-----------------|
| "Não sabemos o CAC" | "Para tudo até saber. É a métrica mais importante." |
| "O produto está bom" | "Bom para quem? O cliente disse isso?" |
| "Estamos crescendo 20%" | "E o LTV? E o churn? Growth sem retenção é ilusão." |
| "Campanha nova" | "Qual hipótese estamos testando? Como vamos medir?" |
