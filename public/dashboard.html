Conectar Claude Code do Bruno ao Nexus
PENDENTE
Para o Bruno poder usar os setores da Nexus diretamente no Claude Code dele, ele precisa rodar um único comando no terminal do computador dele.
1
Abrir o terminal no computador dele (Prompt de Comando ou PowerShell)
2
Rodar o comando abaixo — ele cadastra o Nexus como ferramenta nativa:
claude mcp add nexus-control https://claude-code-febra.onrender.com/mcp/sse --header "x-api-key: nexus-primo-2026"
3
Pronto. Na próxima conversa do Claude Code dele, as ferramentas nexus_chat, nexus_consenso, nexus_conselho, nexus_metricas e nexus_produtos_brn aparecem automaticamente.
⚠️ O servidor pode demorar ~30 segundos para responder na primeira vez do dia (Render gratuito entra em modo de espera). Se der erro, tente de novo após 1 minuto.
📊
Configurar UTMify (fazer junto com Bruno)
AGUARDANDO
Para ver dados de tráfego pago aqui no dashboard, o Bruno precisa adicionar a chave de API do UTMify nas configurações do servidor.
1
No painel do UTMify → Configurações → API → copiar a chave
2
No Render (render.com) → serviço claude-code-febra → Environment → adicionar variável:
UTMIFY_API_KEY = colar a chave aqui
3
Clicar em Save Changes no Render — o servidor reinicia e a integração ativa automaticamente
🔑
Trocar a chave de acesso do MCP (opcional)
OPCIONAL
A chave atual (nexus-primo-2026) é pública. Para segurança, troque por uma senha sua no Render.
MCP_API_KEY = sua-senha-secreta-aqui
Depois de trocar, o Bruno precisa rodar o mesmo comando acima substituindo nexus-primo-2026 pela nova senha.
